## 🔨 Customize

Before loading the extension, you can customize it to fit your needs:

### manifest.json
Edit:
- `name` – Extension name
- `description` – Extension description
- `matches` – Websites where the extension will run

### content.js
Edit:
- Messages and displayed text
- Account information
- Other hardcoded values used by the extension

### icon.png
Replace `icon.png` with your own icon to personalize the extension's appearance in the browser toolbar and extension manager.

> **Note:** After making changes, reload the extension in your browser to apply the updates.
